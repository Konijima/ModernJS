## Overview

ModernJS provides a comprehensive set of features for building modern web applications:

### Navigation & Routing
The [Router](./router.md) provides Angular-inspired routing with guards, lazy loading, and nested routes.

### Data Management
[Forms](./forms.md) offer reactive form handling with built-in validation, while the [HTTP Client](./http.md) provides a reactive approach to API communication.

### User Experience
Enhance your applications with [Animations](./animations.md) for smooth transitions, [Internationalization](./i18n.md) for global reach, and [Modal](./modal.md) dialogs for user interactions.

Each feature is designed to work seamlessly with the core component system while remaining optional - use only what you need.
